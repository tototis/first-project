Linguagens da programação

A linguagem da progração é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software). Um software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução. As principais linguagens são:

  1. JavaScript (executada no client-side e no server-side)
  2. Python
  3. Java
  4. PHP
  5. C#
E o HTML, é uma linguagem de programação?

Não!!! HTML é a sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto. O HTML é o componente base da web, ou seja, ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos. O navegador interpreta textos delimitados por nomes (delimitadores) que têm funções próprias e diferentes.

As tags e seus atributos

A tag HTML de conteúdo é uma marca que define o conteúdo de um documento da respectiva linguagem. Ela envolve o texto e os elementos que aparecem no documento. Tecnicamente, elas são usadas para determinar o tipo de conteúdo que será exibido na página. Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

Os atributos que aprendemos são para textos, listas e links.

Os principais para textos são:

  • h1, h2,h3, h4, h5, h6, p, strong, mark, small, i, u, hr, sub, sup, blockquote
    • Os "h's" são utilizados como subtítulos, sendo o h1 o com a fonte maior e o h6 o com a fonte menor;
    • O p é utilizado para indicar parágrafo;
    • O strong para deixar a fonte em negrito;
    • O mark para dar efeito de marca-texto na fonte;
    • O small para diminuir o tamanho da fonte;
    • O i para deixar a fonte em itálico;
    • O u para deixar a fonte sublinhada;
    • O hr para dividir os textos com uma linha;
    • O blockquote para incluir um texto a parte do parágrafo, geralmente utlizado para citações;
    • O sub e o sup para, na mesma linha, incluir um texto um pouco mais acima ou abaixo do que já foi escrito, como o caso de expoentes nos números.

Os principais para listas são:

  • ul, ol,li
    • O ul é utilizado para indicar uma lista não ordenada por números;
    • O ol é utilizado para indicar uma lista ordenada por números;
    • O li é utilizado para indicar os elementos da lista seja ela ordenada por números ou não.

O principal para links é o:

  • a
    • O a é utilizado para indicar um link. É um elemento de âncora e deve-se definir seu atributo href para que aponte para o URL o qual você quer associar.
